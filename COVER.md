## 📸CodeSnap.nvim

CodeSnap.nvim is a screenshot tool with rich features,
allowing you to capture beautiful code images with ease.

Spotlight:
- Copy snapshot directly to clipboard
- Save snapshot as SVG, PNG or HTML
- Customize themes, fonts, and more
- ⭐ ASCII art snapshot support
